# B_CMS 프로젝트
- 아무리 리액트를 좋아한다하지만 
- 실무에서 죽도록 쓰는 vue로도 개인프로젝트 기록을 남겨보자

# 목표
- B2B 플랫폼에서 자주쓰이는 형태의 CMS 제작

# 완료 기능
1. 로그인
2. 라우터가드
3. 다이나믹라우팅
4. 탭 (캐싱/닫기/새로고침)
5. 메뉴 검색
6. 테마 프리셋(DasyUI Themes / https://daisyui.com/docs/themes/)
7. 서버사이드 i18n / 언어 선택기

# 진행 중
1. ag-grid에 daisyUI theme적용
2. svg파일 기준 메뉴 정보에 맞는 아이콘 메뉴 아이템에 표시

# 진행 목표
